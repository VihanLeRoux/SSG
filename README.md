# Web Assembly SSG

This is a project that will be used to create a static site generator (SSG). The end product will be similar to [Obsidian](https://obsidian.md/) in its base functionality, but will be implemented in stages based on the core use case of serving static web pages.

## Goals

The goals for this project is as follows:

 - Create a system to dynamically create HTML pages.
 - Create an interactive single page application based on React.
 - 

## Technology Stack

The technologies and/or programming languages that will be used in this project are as follows:

 - Web Assembly
 - C++
 - TypeScript
